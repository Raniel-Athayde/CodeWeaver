Como Executar o CodeWeaver (Microserviços)

Abra 3 terminais separados.

# Terminal 1 (Serviço de Análise)
cd analyzer
python app.py

# Terminal 2 (Serviço de Notificação)
cd notifier
python app.py

# Terminal 3 (Gateway Principal)
cd gateway
python app.py

No Navegador:
Acesse: http://localhost:5000

Nota: O Gateway se comunica com os outros serviços via porta 5001 e 5002.
